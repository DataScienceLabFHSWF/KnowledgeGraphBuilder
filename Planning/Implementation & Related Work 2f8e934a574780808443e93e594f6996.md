# Implementation & Related Work

## 1. Implementation Details

### Tech Stack (Example)

- **KG**: Neo4j / RDF store
- **Ontology**: OWL + SHACL
- **Vector DB**: FAISS / Weaviate / PSQL
- **LLMs**: DeepSeek, QWEN, LLaMa
- **Graph ML**: PyTorch Geometric (GraphSAGE)
- **Graph Analysis:** networkX

### Agent Orchestration

- LangGraph / custom DAG
- Message-based interfaces
- Full logging

### Observability

- Every question
- Every source
- Every triple
- Every validation result
